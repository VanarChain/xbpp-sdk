# Changelog

## 0.1.0-beta.1

Initial beta release of the xBPP SDK.

### Features

- Core `evaluate()` function with 12 policy checks
- `wrap()` function for intercepting fetch/x402 payment calls
- Three policy presets: `aggressive`, `balanced`, `riskAverse`
- `BlockedError` and `EscalateError` custom error classes
- Full TypeScript type definitions
